# vue3-vite-admin-template

## 介绍
- 一个非常简洁的后台模板,任何功能都可以自由组织

## 技术栈
- Vue3
- VueRouter
- Pinia
- Sass
- element-plus
- animate.css


## 内部组件
### Edit.vue
- 编辑组件,支持修改与回调

### Loading.vue
- 一个简单的加载组件，你可以随意修改

### Pagination.vue
- 分页组件，进一步封装el的组件

### Table.vue
- 表格组件, 进一步封装el的组件
- 集成了基本的增删改封装

## 开发
### 安装依赖
```sh
npm install
```
### 运行
```sh
npm run dev
```
### 打包
```sh
npm run build
```
